# Stack_Overflow-searcher

Stack Overflow searcherは検索ボックスに入力された文字列をGoogle検索し、そこからStack Overflowのサイトだけを抽出して表示します。

#　使用技術
・　Rust 1.49
・actix-web 3.3.2
・Docker-compose


#　機能一覧
・検索機能
・検索履歴の保存
・履歴からの補完機能
・履歴一括削除
・特定の履歴を一つだけ削除する機能
